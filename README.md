# 大学生でもできる家計簿アプリ

ターミナルで動くシンプルな家計簿アプリです。
Python学習の最初のプロジェクトとして、機能を1つずつ追加しながら作りました。

## 機能

- 支出の記録（品名・金額・日付を自動で保存）
- 支出の一覧表示と合計
- 今月の合計だけを表示
- 支出の削除（番号指定）
- データはJSONファイルに保存されるので、終了しても消えません

## 動かし方

Python 3 が入っていれば、追加のインストールは不要です。

```bash
python3 kakeibo.py
```

メニューが表示されるので、番号を選んで操作します。

```
[1] 支出を追加  [2] 一覧を見る  [3] 今月の合計  [4] 削除  [5] 終了
```

## テストの実行

計算・削除ロジックの自動テストが付いています。

```bash
python3 test_kakeibo.py
```

「すべてのテストに合格しました！」と出れば正常です。

## 使用技術

- Python 3（標準ライブラリのみ）
- データ保存：JSON
